# FireBot
FireBot is a mobile application developed with Flutter to enhance fire safety awareness and emergency response. The app is designed to provide users with real-time information, alerts, and guidance during fire incidents.

Features

Emergency Alerts: Notify users about nearby fire hazards.

Safety Tips: Provides actionable fire safety tips and preventive measures.

Incident Reporting: Users can report fire incidents with location details.

Interactive UI: Built with Flutter for smooth, cross-platform experience.

Persistent Storage: Uses SharedPreferences to store user preferences and data locally.

Technology Stack

Flutter (Dart)

SharedPreferences for local storage

Android/iOS compatible

Purpose

FireBot aims to:

Increase fire safety awareness

Help users respond quickly and appropriately during fire emergencies

Demonstrate practical use of Flutter for real-time, interactive mobile applications

Potential Improvements

Integrate push notifications for real-time alerts

Add map integration to visualize fire locations

Connect to IoT fire sensors for automated updates
